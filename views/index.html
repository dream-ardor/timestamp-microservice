Timestamp Microservice

How to use :

  1. The API endpoint is https://platinum-caribou.glitch.me/api/timestamp/:date_string
  2. A valid date string obey the format as natural date or unixstamp as below, and then api returns a JSON having the structure: e.g. {"unix": 1479663089000 ,"utc": "Sun, 20 Nov 2016 17:31:29 GMT"}
    • Natural Date Example:
    • 2016-12-02
    • 02-12-2016
    • December 2, 2016
    • December 2 2016
    • Dec 2, 2016
    • Dec 2 2016
    • 2 December 2016
    • 2 December 16
    • 2 Dec 2016
    • 2 Dec 16
    • Unixtimestamp Example:
    • 1450137600
  3. if input a invalid date string or empty, the api returns a JSON: {"unix": null, "utc" : "Invalid Date" }

Example Usage:

  • https://platinum-caribou.glitch.me/api/timestamp/2015-12-25
  • https://platinum-caribou.glitch.me/api/timestamp/1450137600

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
